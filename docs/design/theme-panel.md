Teek 提供了一套全新的系统级色彩体系，搭配 Element Plus 的产品级色彩体系，能够满足大部分后台管理系统和数据可视化场景的需求。

所有主题色均通过 CSS 变量暴露，便于在项目中灵活调用和自定义扩展。

::: tip
Teek 提供的功能色会覆盖 Element Plus 的功能色。
:::
