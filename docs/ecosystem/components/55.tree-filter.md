## 基础使用

`defaultValue` 可以在初始化时，默认选择某一个数节点。

监听 `change` 事件，获取当前选中的节点。

::: demo
tree-filter/basic
:::

## API 获取数据

通过 `request-api` 配置项，可以让组件内部调用接口获取数据，并自动添加 `全部` 选项在第一级。

通过 `defaultRequestParams` 配置项，可以设置接口需要传递的参数。

通过 `transformData` 配置项，可以在通过接口获取数据进行加工处理，最后返回一个新的数据。
